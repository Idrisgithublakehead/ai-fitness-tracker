# AI Fitness Tracker

A web-based fitness tracking application that logs workouts and generates training recommendations using modular AI-style logic. The project is designed to be extended with a real AI API backend.

Live Demo: https://idrisgithublakehead.github.io/ai-fitness-tracker/

## Features
- Log workouts (exercise, sets, reps, weight)
- Workout history stored in the browser (localStorage)
- Training recommendation engine (MVP logic)
- Responsive dark UI with card-based layout

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

This project is an active work-in-progress.  
Upcoming improvements include UI refinements and real AI integration.
